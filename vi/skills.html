Giới thiệu Kỹ năng Cá nhân

Chi tiết các kỹ năng nổi bật của tôi trong lĩnh vực lập trình và làm việc

Kỹ năng lập trình

  • HTML / CSS
    75%

    - Nắm vững cấu trúc cơ bản

    - Khả năng tạo trang web theo yêu cầu cụ thể.

  • JavaScript
    65%

    - Thành thạo kết hợp với Html / Css để tạo trang web động

  • PHP
    75%

    - Kinh nghiệm làm việc với MySQL.

    - Xử lý dữ liệu người dùng, server.

  • C++
    80%

    - Sử dụng cấu trúc dữ liệu và thuật toán cơ bản.

    - Nắm vững lập trình hướng đối tượng.

  • C#
    70%

    - Phát triển ứng dụng với .NET Framework.

  • NodeJS
    55%

    - Xây dựng API và ứng dụng thời gian thực.

  • Python
    50%

    - Phân tích dữ liệu và xây dựng các công cụ tự động hóa.

Kỹ năng khác

  • Làm việc nhóm

    - Kinh nghiệm làm việc trong các dự án nhóm Agile/Scrum.

    - Sử dụng thành thạo công cụ quản lý dự án như Jira, Trello.

  • Làm việc độc lập

    - Khả năng quản lý thời gian và hoàn thành công việc đúng hạn.

    - Sáng tạo trong việc giải quyết các vấn đề phức tạp.

  • Khả năng tư duy logic

    - Kỹ năng phân tích vấn đề và đưa ra các giải pháp tối ưu.

    - Hiểu rõ các thuật toán và cấu trúc dữ liệu cơ bản.

  • Phân tích, giải quyết vấn đề

    - Kinh nghiệm giải quyết sự cố.

    - Sử dụng các công cụ debug như DevTools, Visual Studio Code, Visual Studio.
